# Governance Drafts

These files are non-final governance drafts for M12 submission preparation.
They document prepared decisions, not final approvals.

The files in this directory must not be used to pass final submission gates:

- `deadline_verification.draft.json`
- `repository_policy_decision.draft.json`
- `author_final_approval.draft.json`

Final gate files must be created only after human verification and must live in
`submission/escience2026/` with these exact names:

- `deadline_verification.json`
- `repository_policy_decision.json`
- `author_final_approval.json`

Conservative deadline planning is not final deadline verification. A repository
policy draft is not a final repository decision. An author final approval draft
is not a signature. `final_submission_ready` remains false until the final gate
files exist and pass the submission linter.
