# License Decision Support

The repository, manuscript, and submission artifacts do not yet have a final
license decision for eScience submission and possible artifact release. This
file is decision support only; it is not the final license decision.

## Options

1. MIT for code + CC-BY-4.0 for manuscript/artifacts.
2. Apache-2.0 for code + CC-BY-4.0 for manuscript/artifacts.
3. Custom institutional license.
4. `undecided`.

## Required Human Decision

Human authors must choose the code, manuscript, and artifact licenses before
final submission or public artifact release. The final gate file
`submission/escience2026/license_decision.json` must be created only after that
decision is made.

Current state: `license_status = undecided`.
