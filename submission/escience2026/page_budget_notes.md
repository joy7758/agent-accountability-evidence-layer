# Page Budget Notes

The venue linter uses a rough estimate:

```text
word_count / 620 + table_rows / 25 + 0.25 pages per markdown figure marker
```

This estimate is only a readiness signal. Human authors must check the final
IEEE template. The full paper target from the M9 brief is 8 pages excluding
references.

Suggested compression if over budget:

- Merge Problem and Requirements if needed.
- Move detailed tool descriptions to an artifact appendix if allowed.
- Keep only one pipeline figure and one compact evaluation table.
- Shorten related work into positioning paragraphs.
- Preserve Limitations and AI-use disclosure even under page pressure.
