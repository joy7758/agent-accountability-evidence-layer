# Contribution Box

1. ASIEP defines a minimal FAIR evidence object profile for auditable agent
   self-improvement events.
2. The repository implements an agent-native conformance and evidence pipeline:
   validator, repairer, resolver, importer, packager, evaluator, paper linter,
   citation linter, and venue linter.
3. ASIEP maps local evidence objects to PROV, OTel-like traces,
   LangSmith-like traces, FDO-like records, and RO-Crate-like metadata with
   explicit lossiness.
4. The local evaluation reports reproducible fixture-based checks over valid
   examples, attack cases, evidence bundles, generated packages, and paper
   evidence assets.
